# IDE (Intensional Dynamics Engine)

【IDEとは？】 M-Tokuniが開発した、システム動作を滑らかに保つための「物理制御エンジン」です。 PCやブラウザにかかる高い負荷を、まるで「物理的なバネ」のようにしなやかに吸収・分散し、フリーズやカクつきを抑えることができます。

🚀 まずは体験する（実証デモ）
IDEを導入すると「何がどう変わるのか」を一瞬で体験できるサンプルです。ブラウザでそのまま開くだけで、従来型制御とIDEの圧倒的な違いを体感いただけます。

フリーズ防止の実証デモ (examples/HAN_Micro-POC_01.html)

※ 意図的に超高負荷をかけ、システムの「粘り強さ」を比較する実験コードです。

🛠 使いかた
ご自身のプログラムに HannyaPhysicsCore を組み込み、制御したい要素の「歪み（Violation）」を定義するだけで物理制御が開始されます。 具体的な実装方法は、examples/ フォルダ内のコード を参考にしてください。中身が見えるオープンなソースコードとして公開しています。
