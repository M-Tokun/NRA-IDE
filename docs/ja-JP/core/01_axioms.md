# **0. Purpose of This Chapter**

本章では、NRA‑IDE を成立させる **Nomological Ring Axioms（公理系）** を定義する。  
これらは「守った方が良いルール」ではなく、  
**破ると構造が即座に破断する“物理法則レベルの不変条件”**である。

NRA‑IDE は意味を扱わないため、  
公理はすべて **因果構造（Cause）** に基づいて定義される。

---

# **1. Nomological Ring（律環）とは何か**

Nomological Ring（律環）は、  
NRA‑IDE の基盤となる **因果の循環構造**である。

特徴：

- **中心が存在しない（No Target Reference Point）**  
- **意味を持たない**  
- **因果量（δ, τ, R, ω）が循環する**  
- **Cause → Effect の一方向性が維持される**

この構造は、物理学の「散逸構造」や  
アニメーションの「非停止系の動き」に近い。

---

# **2. 公理一覧（Axioms）**

NRA‑IDE は以下の **3つの公理**によって成立する。

---

## **Axiom 1 — No Target Reference Point（中心不在の公理）**

**NRA‑IDE には“中心”が存在しない。**

つまり：

- 距離  
- 類似度  
- スコア  
- 最適化目標  
- 正解ラベル  

これらは **構造的に定義不可能**。

### **理由（構造的根拠）**

中心を定義すると：

- すべての構造量が「中心からの距離」に変換される  
- 意味ベースの最適化が発生する  
- Goodhart’s Law が発動する  
- 構造が破断する

NRA‑IDE は **中心を持たない散逸構造**であるため、  
距離・スコアは存在しない。

---

## **Axiom 2 — Causal Diode（因果ダイオードの公理）**

**Cause → Effect の一方向性は絶対である。  
Effect → Cause（Π⁻¹）は禁止される。**

### **Reverse Projection（Π⁻¹）とは何か**

Effect（出力）を Cause（入力）に逆流させる行為：

- 過去出力を参照する  
- 意味で改善する  
- スコアで最適化する  
- 文脈を意味で補完する  

これらはすべて **Reverse Projection**。

### **破断理由**

Reverse Projection が起きると：

- 因果の向きが崩れる  
- 構造が閉じず、循環が壊れる  
- R（破断比率）が急上昇  
- Fail‑Closed が発火する

---

## **Axiom 3 — Semantic Prohibition（意味禁止の公理）**

**NRA‑IDE は意味を扱わない。  
意味処理は Reverse Projection を誘発するため禁止される。**

### **意味処理の例**

- 危険かどうかを意味で判断  
- 文脈を意味で解釈  
- 意図を推測  
- 自然さを評価  

これらはすべて **Effect 側の処理**であり、  
Cause 側の構造を破壊する。

### **破断理由**

意味処理は：

- 過去出力を参照する  
- 意味的最適化を行う  
- 中心を暗黙に導入する  
- 距離・スコアを生成する  

結果として **Axiom 1 と 2 を同時に破る**。

---

# **3. 公理が守られたときの構造的性質**

公理 1〜3 が守られると、  
Nomological Ring は以下の性質を持つ。

### **3.1 非停止系（ω > 0）**
構造が回転し続ける。  
止まる（ω = 0）と破断。

### **3.2 散逸構造（δ と τ の呼吸）**
ゆらぎ（δ）と厚み（τ）が呼吸するように変動する。

### **3.3 破断比率 R の安定**
R = δ/τ が 1.0 未満に保たれる。

### **3.4 意味に依存しない安定性**
モデル・言語・文脈に依存しない。

---

# **4. 公理違反が起きたときの挙動**

公理違反は **即座に構造破断**を引き起こす。

| 公理違反 | 破断の原因 | 結果 |
|---------|------------|------|
| Axiom 1（中心導入） | 距離・スコア生成 | 最適化暴走 |
| Axiom 2（逆流） | Reverse Projection | 因果崩壊 |
| Axiom 3（意味処理） | 意味補完 → 逆流 | Fail‑Closed |

破断時は必ず **Fail‑Closed → 沈黙 → Discard Log** の順で処理される。

---

# **5. Summary（まとめ）**

Nomological Ring Axioms の本質は次の一文に集約される。

> **「NRA‑IDE は、中心を持たず、意味を扱わず、因果の一方向性だけで成立する。」**

そしてもう一つ。

> **「公理違反は“エラー”ではなく、構造破断である。」**
