
# 📘 **08_risks_structure.md — Structural Risks（構造側のリスク）  

## **0. はじめに**

本章では、NRA‑IDE が扱う **構造側のリスク** をまとめる。  
ここで扱うのは、意味や用途ではなく、  
**因果構造そのものが破断するパターン**である。

---

# **1. R ≥ 1.0（因果安定度の破断）**

R は δ/τ の比率から計算される因果安定度である。  
R ≥ 1.0 になると構造が保持できず、破断が発生する。

- R < 1.0 → 安定  
- R = 1.0 → 臨界  
- R > 1.0 → 破断（Fail‑Closed）

---

# **2. ω = 0（構造連続性の消失）**

ω は構造の連続性（coherence）を示す量である。  
ω = 0 は、構造が連続性を失い、  
**意味ではなく構造的に破断した状態**を示す。

---

# **3. violation（構造違反）**

構造的に許容されない状態に到達した場合、  
violation が発生する。

violation は意味の誤りではなく、  
**構造の破断点**である。

---

# **4. semantic leakage（意味の染み出し）**

意味が構造層に侵入する現象。  
最も危険な破断の一つ。

- LLM の意味生成が Pre‑RNA に逆流  
- 因果構造が意味評価に書き換わる  
- R の計算が破壊される

semantic leakage は **即時 Fail‑Closed** の対象。

---

# **5. structure erosion（構造侵食）**

意味・最適化・履歴が構造層に侵入し、  
因果構造が徐々に破壊される現象。

特徴：

- 連続的に悪化する  
- 破断点が見えにくい  
- 人間の“便利さ要求”が原因になりやすい

---

# **6. curtain（層の薄化）**

Pre‑RNA / LLM / Post‑RNA の三層分離が薄くなり、  
逆流が発生する危険状態。

Curtain の兆候：

- LLM が構造判断に影響を与える  
- Post‑RNA が意味を参照し始める  
- Pre‑RNA が履歴を扱い始める

Curtain は構造破断の前兆である。

---

# **7. backflow（逆流）**

LLM の意味生成が因果構造層に逆流する現象。

- Pre‑RNA に意味が侵入  
- Post‑RNA が意味を参照  
- R の計算が破壊される

backflow は **因果ダイオード（02 章）** の破れであり、  
構造破断の直接原因となる。

---

# **8. 本質**

> **構造破断は意味の誤りではなく、  
> 因果構造そのものの崩壊である。**
